## din

- DIN 字体家族源自德国的工业标准字体，广泛应用于工业、交通标识以及各种设计领域。
- 起源于 1930 年代德国的工业标准字体（DIN 1451），DIN 是 "Deutsches Institut für Normung"（德国标准化学会）的缩写。
- [DIN 1451](https://en.wikipedia.org/wiki/DIN_1451)
