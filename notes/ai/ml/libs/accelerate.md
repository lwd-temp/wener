# accelerate

- [huggingface/accelerate](https://github.com/huggingface/accelerate)
  - Apache-2.0, Python
  - pip:accelerate
  - by Hugging Face
  - 简化和加速深度学习模型在不同硬件环境（如 CPU、GPU、TPU）上的分布式训练和推理

```bash
# $HOME/.cache/huggingface/accelerate/default_config.yaml
accelerate config
```
